Reglas de Seguridad

Las reglas de seguridad de Firestore deben permitir acceso de lectura y escritura para que la aplicación funcione correctamente.

Reglas recomendadas para desarrollo (NO usar en producción):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Para establecer las reglas, ve a la Consola de Firebase → Tu proyecto → Firestore Database → Reglas
